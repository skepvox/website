# Mapa

<DemosMapa />

::: info Nota editorial (Skepvox)
Este mapa é uma visualização **técnica** do grafo do Skepvox (nós e conexões).
Ele não “prova” nada por si só: cada conexão deve ser interpretada junto das
fontes e do status descritos nas notas.

Dicas:

- Clique/toque em um nó para selecionar e ver detalhes.
- Toque/clique no mesmo nó novamente (ou use “Abrir nota”) para abrir a página.
- Arraste nós para reorganizar.
- Use scroll/trackpad para zoom.
:::
