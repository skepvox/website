# Imagens de perfil — Pessoas (`/demos`)

Coloque aqui as imagens de perfil usadas nas notas de pessoas em `/demos/pessoas/`.

Convenção recomendada:

- Caminho público: `/images/demos/pessoas/<slug>.(png|jpg|webp)`
- `<slug>` = nome do arquivo da nota (sem `.md`), ex.: `luiz-inacio-lula-da-silva`
- Formato: quadrado, 512×512+ (ideal), foco no rosto e sem dados pessoais

Após adicionar/atualizar retratos, gere os SVGs de pontilhismo:

- `pnpm demos:portraits`
